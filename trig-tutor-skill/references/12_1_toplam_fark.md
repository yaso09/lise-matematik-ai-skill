# 12. Sınıf Trig-1: Yay Toplam/Fark ve Yarım Açı Bağıntıları

## Toplam ve Fark Formülleri
```
sin(a+b) = sina·cosb + cosa·sinb
sin(a−b) = sina·cosb − cosa·sinb

cos(a+b) = cosa·cosb − sina·sinb
cos(a−b) = cosa·cosb + sina·sinb

tan(a+b) = (tana + tanb) / (1 − tana·tanb)
tan(a−b) = (tana − tanb) / (1 + tana·tanb)

cot(a+b) = (cota·cotb − 1) / (cota + cotb)
```

## Yarım Açı (Çift Açı) Bağıntıları
```
sin(2x) = 2·sinx·cosx

cos(2x) = cos²x − sin²x
        = 1 − 2·sin²x
        = 2·cos²x − 1

tan(2x) = 2·tanx / (1 − tan²x)

cot(2x) = (cot²x − 1) / (2·cotx)
```

## Yardımcı Formüller (bunlardan türetilebilir)
```
sin²x = (1 − cos2x) / 2
cos²x = (1 + cos2x) / 2
```

## Sık Kullanılan Değerler (toplam ile türetilir)
```
sin15° = sin(45°−30°) = (√6−√2)/4
cos15° = cos(45°−30°) = (√6+√2)/4
tan15° = 2 − √3

sin75° = cos15° = (√6+√2)/4
cos75° = sin15° = (√6−√2)/4
tan75° = 2 + √3
```

## Çarpım → Toplam Formülleri
```
2·sina·cosb = sin(a+b) + sin(a−b)
2·cosa·cosb = cos(a+b) + cos(a−b)
2·sina·sinb = cos(a−b) − cos(a+b)
```

## Toplam → Çarpım Formülleri
```
sinx + siny = 2·sin((x+y)/2)·cos((x−y)/2)
sinx − siny = 2·cos((x+y)/2)·sin((x−y)/2)
cosx + cosy = 2·cos((x+y)/2)·cos((x−y)/2)
cosx − cosy = −2·sin((x+y)/2)·sin((x−y)/2)
```

## Üçgen Uygulaması
ABC üçgeninde A + B + C = 180° olduğundan:
- sin(A+B) = sinC,  cos(A+B) = −cosC
- cos A·cosB − sinA·sinB = cos(A+B) = −cosC
