# 11. Sınıf Trig-3: Sinüs ve Kosinüs Teoremi

## Kosinüs Teoremi
ABC üçgeninde a, b, c kenar uzunlukları; A, B, C karşı açılar:

```
a² = b² + c² − 2bc·cosA
b² = a² + c² − 2ac·cosB
c² = a² + b² − 2ab·cosC
```

**Ne zaman kullanılır?**
- 3 kenar biliniyorsa (açı bulmak için)
- 2 kenar + aradaki açı biliniyorsa (3. kenarı bulmak için)

**Özel durum:** a² = b² + c² − bc ise cosA = 1/2 → A = 60°

## Sinüs Teoremi
```
a/sinA = b/sinB = c/sinC = 2R
```
R = çevrel çemberin yarıçapı

**Ne zaman kullanılır?**
- 1 kenar + 2 açı biliniyorsa
- 2 kenar + bunlardan birine karşı açı biliniyorsa (dikkat: iki çözüm olabilir)

## Alan Formülü
```
Alan = (1/2)·a·b·sinC = (1/2)·b·c·sinA = (1/2)·a·c·sinB
```

## Sık Sorulan Soru Tipleri
- En büyük açı en büyük kenara karşıdır → kosinüs teoremi ile bul
- Üçgenin var olma koşulu: her açı > 0, açılar toplamı 180°
- m(Â) = 120° → cosA = −1/2
- m(Â) = 30° → sinA = 1/2, cosA = √3/2
